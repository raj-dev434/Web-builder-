## Description
 This is an web builder tools Majorily Focused on Movie Site Building .Users can instantly build an website with drag and drop .
## Features
This web builder has an inbuilt feature for Netflix component which help us users  to instantly create netflix related website just by drag and drop
This web builder has an other component such as Layout,Media,Element and basic component as well which used to build normal website instantly
This web builder has an inbuilt tools which help user to change color,font,size .user also able to adjust the entire Container.
This web builder has an integrated api feature which helps user to create any movie related website instantly
